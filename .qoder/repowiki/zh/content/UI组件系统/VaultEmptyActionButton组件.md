## 简介

VaultEmptyActionButton是Photo Vault应用中的一个专用按钮组件，专门用于保险库(empty state)界面中。该组件为用户提供清晰的操作引导，支持主要操作和次要操作两种类型，具有加载状态指示和响应式交互反馈。

## 组件概述

VaultEmptyActionButton是一个自定义的Compose按钮组件，专为保险库空状态界面设计。它继承了Material Design的设计原则，同时融入了应用的品牌色彩和交互规范。

### 主要用途
- 在保险库为空时提供操作引导
- 支持主要操作（如导入照片）
- 支持次要操作（如拍摄私密照片）
- 提供加载状态反馈

### 技术特点
- 基于Jetpack Compose构建
- 支持主次两种按钮样式
- 内置加载状态指示器
- 响应式触摸反馈
- 自适应主题色彩

## 核心功能特性

### 1. 双模式设计
组件支持两种操作模式：
- **主要按钮**：用于执行关键操作，如导入照片
- **次要按钮**：用于辅助操作，如拍摄私密照片

### 2. 加载状态管理
- 当操作正在进行时显示圆形进度指示器
- 禁用按钮交互以防止重复操作
- 使用品牌色彩确保视觉一致性

### 3. 响应式交互
- 集成触摸反馈系统
- 支持防抖动点击处理
- 提供视觉和触觉反馈

### 4. 主题适配
- 自动适配深色/浅色主题
- 使用统一的颜色令牌系统
- 符合Material Design规范

## 设计规范

### 尺寸规格
组件采用固定高度设计，确保在不同屏幕尺寸下的一致性：

| 规格项 | 数值 | 描述 |
|--------|------|------|
| 高度 | 52dp | 固定按钮高度 |
| 圆角半径 | 26dp | 半径等于高度的一半 |
| 内边距 | 16dp | 左右内边距 |
| 图标尺寸 | 18dp | 图标大小 |
| 文本字体 | 15sp | 字体大小 |

### 颜色系统
组件使用统一的颜色令牌系统，确保视觉一致性：

- **主要按钮容器**：`UiColors.Button.primaryContainer` (#4A9EFF)
- **次要按钮容器**：`UiColors.Button.secondaryContainer` (#1A202C)
- **主要按钮文字**：`UiColors.Button.primaryContent` (白色)
- **次要按钮文字**：`UiColors.Button.secondaryContent` (#EAF1FF)

### 间距规范
- 上下间距：12dp
- 图标与文字间距：8dp
- 宽度：全宽（fillMaxWidth）

## 实现细节

### 组件签名
```kotlin
@Composable
private fun VaultEmptyActionButton(
    text: String,
    onClick: () -> Unit,
    isPrimary: Boolean,
    modifier: Modifier = Modifier,
    loading: Boolean = false
)
```

### 核心实现逻辑

#### 1. 基础布局结构
组件采用Row布局，居中对齐内容，确保文字和图标的完美对齐。

#### 2. 条件渲染
- **主要按钮且非加载状态**：显示左侧图标 + 文字
- **加载状态**：显示圆形进度指示器
- **其他情况**：显示纯文字

#### 3. 交互处理
- 使用`rememberFeedbackInteractionSource()`创建反馈源
- 应用`pressFeedback`和`throttledClickable`修饰符
- 根据loading状态禁用点击事件

#### 4. 样式应用
- 动态设置背景颜色（主/次按钮）
- 应用圆角形状（RoundedCornerShape）
- 设置固定高度和内边距

## 故障排除指南

### 常见问题及解决方案

#### 1. 按钮不响应点击
**症状**：点击无反应
**可能原因**：
- loading参数设置为true
- onClick回调为null
- 父级容器禁用了交互

**解决方法**：
- 检查loading状态
- 确保onClick回调有效
- 验证父级容器的交互设置

#### 2. 视觉样式异常
**症状**：颜色或尺寸不符合预期
**可能原因**：
- 主题颜色未正确加载
- 颜色令牌配置错误
- 屏幕密度适配问题

**解决方法**：
- 检查UiTokens配置
- 验证颜色资源文件
- 测试不同屏幕密度

#### 3. 加载状态显示问题
**症状**：进度指示器不显示或显示异常
**可能原因**：
- loading参数传递错误
- 进度指示器颜色配置问题
- 组合逻辑错误

**解决方法**：
- 检查loading状态逻辑
- 验证进度指示器颜色
- 审核条件渲染逻辑
