## 简介
本指南围绕 AI 照片保险库的"私密拍照"能力，提供从专业级相机集成、权限管理、实时预览与拍照流程，到直拍直存、不写入系统相册、实时加密与隐私保护的完整实现说明。文档同时解释与相册管理系统的集成方式、照片元数据处理以及与加密子系统的衔接。

**更新**：私密拍照功能经过重大现代化改进，包括状态持久化优化（rememberSaveable迁移）、设备方向检测系统、设置面板交互增强、直接媒体保存流程、UI界面重构、缩略图生成系统和设置面板增强。新的实现确保在配置变更时状态正确恢复，优化了拍照和录制流程，集成了专业的缩略图生成系统，并提供了更丰富的相机参数配置选项。新增的防重复点击机制和直接媒体保存流程改进进一步提升了用户体验和系统稳定性。

## 核心组件
- **专业级相机界面**：负责权限申请、实时预览绑定、闪光灯与镜头切换、定时拍摄、多倍率缩放、曝光补偿、网格辅助线等专业摄影功能。
- **可折叠设置面板**：提供可折叠的设置面板，支持闪光灯、定时器、网格、曝光、分辨率、帧率等参数配置，使用 rememberSaveable 确保状态持久化。
- **预览确认界面**：提供预览确认界面，支持重拍和保存到保险库操作，集成 VaultProgressiveImage 缩略图生成系统。
- **缩略图生成系统**：集成 VaultProgressiveImage 组件，支持图片和视频缩略图的异步加载和渐进式显示。
- **动态模式切换**：支持拍照模式（PHOTO）和视频模式（VIDEO）之间的无缝切换，每个模式都有独立的控制界面和功能。
- **视频录制系统**：基于 CameraX VideoCapture API 实现 MP4 视频录制，支持实时录制指示器、录制时长显示、录制状态管理。
- **自动相机重新绑定**：当相机出现错误或绑定失败时，自动进行最多3次重试，确保相机功能的稳定性。
- **错误分类系统**：将拍照/录制错误细分为权限、硬件、IO、超时、未知等类别，提供精准的错误诊断和用户提示。
- **ZoomRail 连续滑块**：提供流畅的缩放控制体验，支持0.6x/1x/2x预设和连续滑动调节。
- **自动曝光重置**：支持一键将曝光指数重置为自动模式，简化专业摄影操作。
- **诊断日志系统**：记录相机绑定、预览首帧、拍摄过程、录制过程、内存使用等关键指标，便于问题排查和性能监控。
- **拍照流程**：调用 CameraX ImageCapture 拍照，输出到临时文件，支持预览确认界面，不写入系统相册。
- **视频录制流程**：调用 CameraX VideoCapture 进行 MP4 录制，输出到保险库目录，支持录制状态管理和错误处理。
- **保险库存储**：在应用私有目录按相册组织，支持默认相册与迁移逻辑。
- **加密与密钥**：使用 Android Keystore 的 AES-256-CBC 密钥，前置 IV 的加密格式。
- **元数据与索引**：通过 Room 实体记录密文路径、缩略图路径、元数据 JSON、创建时间等。
- **状态持久化优化**：全面迁移到 rememberSaveable，确保配置变更时状态正确恢复。
- **设备方向检测系统**：使用 OrientationEventListener 自动检测设备方向变化，动态调整相机捕获方向，确保图像保持正确方向。
- **设置面板交互增强**：添加点击外部区域自动关闭设置面板的功能，提升用户体验。
- **防重复点击机制**：引入 ThrottledClick 和 pressFeedback 改善用户体验，防止误操作。
- **直接媒体保存流程**：优化临时文件处理和直接保存机制，提升性能和可靠性。

**更新**：私密拍照功能经过重大现代化改进，包括状态持久化优化（rememberSaveable迁移）、设备方向检测系统、设置面板交互增强、直接媒体保存流程、UI界面重构、缩略图生成系统和设置面板增强。新的实现确保在配置变更时状态正确恢复，优化了拍照和录制流程，集成了专业的缩略图生成系统，并提供了更丰富的相机参数配置选项。新增的防重复点击机制和直接媒体保存流程改进进一步提升了用户体验和系统稳定性。

### 设备方向检测系统与自动调整
- **OrientationEventListener 集成**：新增 OrientationEventListener 监听设备方向变化，自动调整相机捕获方向。
- **方向检测逻辑**：根据设备旋转角度（0°、90°、180°、270°）动态设置 ImageCapture 的 targetRotation。
- **自动方向调整**：当设备旋转时，自动更新相机捕获方向，确保拍摄的图像保持正确的方向。
- **生命周期管理**：在 DisposableEffect 中启用和禁用方向监听器，确保资源正确释放。
- **EXIF 方向处理**：结合 EXIF 旋转处理函数，确保图像方向的一致性。

## 性能考虑
- **捕获模式**：使用最小延迟捕获模式，降低拍摄时延。
- **预览缩放**：采用填充中心的 ScaleType，保证全屏预览体验。
- **I/O 线程**：拍照/录制回调与后续加密/写盘在 IO 线程执行，避免阻塞 UI。
- **防重复点击**：捕获状态变量防止并发拍摄/录制导致的异常。
- **闪关模式**：根据开关动态设置，兼顾低光场景与能耗。
- **预览解码优化**：使用合适的采样率解码预览 Bitmap，平衡质量和性能。
- **内存管理**：及时释放临时文件和 Bitmap 资源，监控内存峰值使用。
- **自动相机重新绑定**：在应用前台时自动重新绑定相机，确保相机功能的稳定性。
- **诊断日志性能**：仅在调试模式下记录详细日志，避免影响生产环境性能。
- **视频录制优化**：使用 Recorder 和 QualitySelector 进行高效录制，支持质量降级策略。
- **录制状态管理**：及时清理录制对象和文件，避免内存泄漏。
- **设置面板性能**：可折叠设置面板减少不必要的布局计算，提升界面响应速度。
- **预览确认界面优化**：预览确认界面采用懒加载机制，避免不必要的资源消耗。
- **状态持久化优化**：使用 rememberSaveable 确保配置变更时状态正确恢复，提升应用性能。
- **缩略图生成优化**：VaultProgressiveImage 使用异步加载和缓存机制，提升缩略图显示性能。
- **设备方向检测优化**：OrientationEventListener 仅在设置面板显示时启用，减少不必要的 CPU 占用。
- **外部区域点击优化**：设置面板外部区域点击关闭功能使用高效的点击事件处理机制。
- **防重复点击优化**：ThrottledClick 使用 rememberUpdatedState 优化状态管理，减少不必要的重绘。

## 故障排查指南
- **无相机权限**：检查清单权限声明与运行时授权流程，确保 CAMERA 权限已授予。
- **拍照/录制失败**：检查 ImageCapture/VideoCapture 回调错误类型，定位硬件或资源占用问题。
- **无法预览**：确认 PreviewView 已正确注入 surfaceProvider，生命周期绑定是否生效。
- **加密异常**：核对密钥是否存在且未被系统清理，输入载荷长度是否满足要求。
- **元数据缺失**：确认 Room 表结构与索引是否存在，必要时重建数据库或迁移脚本。
- **定时拍摄异常**：检查定时器状态管理，确保倒计时逻辑正确执行。
- **缩放功能失效**：验证相机支持的缩放范围，确保缩放值在有效范围内。
- **曝光补偿问题**：确认相机支持曝光补偿功能，检查曝光范围获取是否成功。
- **自动重新绑定失败**：检查相机状态监听器是否正常工作，确认重试次数限制。
- **诊断日志问题**：确认 CAMERA_DIAG_TAG 日志标签是否正确配置，检查日志输出级别。
- **内存使用过高**：监控峰值内存使用，检查 Bitmap 解码和临时文件清理。
- **视频录制问题**：检查 VideoCapture 配置，确认 Recorder 和 QualitySelector 设置正确。
