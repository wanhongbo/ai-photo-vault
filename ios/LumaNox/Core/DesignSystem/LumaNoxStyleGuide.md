# LumaNox UI Style Guide

Generated on 2026-05-20 from Android `UiTokens.kt` and iOS `LNColor/LNTokens`.

## Typography

- Display/page title: 24pt bold
- Headline/empty title: 22pt bold
- Section title: 18pt semibold
- Row/card title: 16pt semibold
- Body: 14pt regular
- Large body: 15pt regular
- Label/meta/tab: 12pt medium
- Dialog title/body: 18pt bold / 14pt regular
- PIN page: title 24pt bold, subtitle 14pt, digits 22pt regular

## Radius

- Dialog: 22
- Main card: 20
- Paywall/card variant: 18
- Thumbnail/input: 12
- Settings row: 12
- Icon well/top bar button: 14
- Floating bottom nav: 24

## Core Colors

- Brand blue: #4A9EFF
- Background top/bottom: #0B1324 / #05080D
- Section background: #0C1523
- Dialog background: #101722
- Stroke: #223247
- Title/subtitle: #EAF1FF / #8EA2C0
- Warning gold: #E8C547
- Success: #21C277
- Error: #FF4372

## Component Rules

- Cards use 16pt padding, 20pt radius, 1pt #223247 stroke.
- Rows use 12pt radius, 12pt vertical/horizontal padding, 44pt minimum tap target.
- Dialogs use max width 308pt, 22pt padding, 22pt radius, title/body spacing 11pt, button spacing 12pt.
- Bottom nav is 88pt high, 24pt radius, active item uses blue 12% fill + 1pt blue stroke.
