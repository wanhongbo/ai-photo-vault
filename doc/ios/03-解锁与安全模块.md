## 技术方案要点

### PIN / 图案

- SwiftUI 自定义输入；哈希使用 **CryptoKit SHA256**；敏感状态存 Keychain。

### 生物识别

- `LAContext.evaluatePolicy`；失败回退 PIN；Keychain item 可设 `accessControl` 要求生物识别。

### 后台锁定

- `scenePhase == .background` / `.inactive`（策略与 Android 对齐）触发锁定。

### 伪装图标（付费）

- `UIApplication.shared.setAlternateIconName` + 预置备用图标资源；App Store 审核需合规说明。

### 闯入抓拍

- `AVCaptureSession` + 前置 `AVCaptureDevice`；`AVCapturePhotoOutput` 拍照 → 加密存储；注意摄像头使用说明（Info.plist）。
