## 技术方案要点

### 数据模型

- Core Data / GRDB 与 Android 字段语义对齐，便于备份包恢复。

### 预览

- `ScrollView` + 缩放；或使用 **QLPreviewController** 仅当格式适合；解密策略同临时文件或内存。

### 导出

- `PHPhotoLibrary.performChanges` 创建 `PHAssetCreationRequest`，将解密后的 `UIImage` / 文件写入；完成后删除临时文件。
