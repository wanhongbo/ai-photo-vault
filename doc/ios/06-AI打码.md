## 技术方案要点

### 推理

- 与 Android **同一 `.tflite`** + Metal Delegate，或单独 Core ML（见总览 **第六节**）；输出对齐需同一套后处理参数。

### 图像效果

- Core Image（`CIFilter`）或 `vImage` 对 ROI 模糊/马赛克；注意色彩空间与方向（EXIF）。

### 线程

- `Task.detached` / 专用队列执行推理与像素处理，再 `MainActor` 刷新界面。

## 分层落位（参考）

- **Domain**：与 Android 一致的检测与打码契约；**Data**：TFLite/Core ML 与 Core Image 管线。
