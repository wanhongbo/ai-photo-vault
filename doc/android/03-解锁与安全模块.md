## 技术方案要点

### PIN / 图案

- 自定义采集控件 → 口令经 **SHA-256**（加盐策略双端一致）仅存哈希；存储于加密偏好或 DB。

### 生物识别

- `BiometricPrompt` + `BiometricManager`；可选 `CryptoObject` 与 Keystore 密钥绑定，错误后回退 PIN。

### 后台锁定

- `ProcessLifecycleOwner` / `Lifecycle.Event.ON_STOP`（或按产品选用 `ON_PAUSE`）触发锁屏状态；需处理多窗口与画中画策略。

### 伪装图标（付费）

- 通过 **activity-alias** 或多入口 + `PackageManager.setComponentEnabledSetting` 切换默认启动器图标；注意各 ROM 限制与商店审核对「误导性图标」的规则。

### 闯入抓拍（付费）

- CameraX 前置 **`ImageCapture`** 或 `ImageAnalysis`，连续验证失败达阈值触发；图像走 **02-照片导入与AES本地加密** 同一加密管线入库。
